# RuBQ Quiz Bot

Тестовый квиз-бот с веб-интерфейсом для вопросов из базы данных RuBQ 2.0.

## Требования

- Node.js (версия 14 или выше)
- npm

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

Запустите сервер:
```bash
npm start
```

Или:
```bash
node server.js
```

Сервер запустится на `http://localhost:3000`

Откройте браузер и перейдите по адресу `http://localhost:3000`

**Примечание:** Порт изменен с 5000 на 3000, так как порт 5000 обычно занят системным AirPlay Receiver в macOS.

## Функциональность

- Случайные вопросы из тестового набора RuBQ 2.0
- Проверка ответов с учетом различных вариантов написания
- Подсказки из параграфов Wikipedia (если доступны)
- Статистика: счет, количество вопросов, точность
- Современный интерфейс с Tailwind CSS

## Структура проекта

- `server.js` - Express сервер с API для вопросов
- `index.html` - Веб-интерфейс с Tailwind CDN
- `package.json` - Зависимости проекта
- `RuBQ/` - База данных RuBQ 2.0 (клонирована из GitHub)

## API Endpoints

- `GET /api/question/random` - Получить случайный вопрос
- `GET /api/question/:uid` - Получить вопрос по ID
- `POST /api/check-answer/:uid` - Проверить ответ пользователя
- `GET /api/stats` - Статистика по датасету
